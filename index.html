Syllabus

Expectations and style of class

In this class we will refresh our knowledge of the math we've already learned and use computational skills to apply this knowledge to interesting and important situations you may encounter in life.

Functions and compositions (Algebra review)

Project 1: due Monday, January 29

We review what you've learned about math in a new light and build some useful tools

Calculus review

Project 2: due Monday, February 5

1. Thinking about calc with the motivating examples of e^x and Riemann sums, 2. Limits and continuity, deriving limits for well known polynomials for f(x+h) as h->0, 3. Applications of calc through algebraic ideas we already know

Measurement and geometry review

Project 3: due Monday, February 12

We take a break and have a fun week to learn about units of measure and think about what it means to measure something. (Hint: The word "rational" is related to "ratio", which implies that you can measure and compare things with the same units.)

Combinatorics and Probability... a new topic that will help with the coding part of this class!

Project 4: due Monday, February 19

We investigate ways of counting and apply those techniques to study probability.

Installing Python

Installation: due Monday, February 26

We're meeting in Mrs. U's computer lab this week... don't try to install on your own yet, we'll go over this together!

Mid-semester review and discussion of the final project

Proposal: due Monday, March 4

Make sure to meet with me to discuss your interests/ideas for the final project before Spring break! We haven't gotten to too much coding but I can help tell you what sort of final project ideas are possible in the time we have.

Rock, paper, scissors!

Project 7: due Monday, March 25

We create the game rock, paper, scissors in Python.

Slicing

Project 8: due Monday, April 1

We study ways to manipulate text and do tokenizing in Python.

Making the power function

Project 9: due Monday, April 8

We create a loop-based power function that inputs a numeric value (base) b, a nonnegative integer (power) p and outputs the value b**p.

More on functions... and doing algebra in Python to solve data analysis problems

Project 10: due Monday, April 22

We begin to build the toolkit for more advanced data analysis!

Regularization Project

Project 11: due Monday, May 6

We study more advanced ways to draw lines of best fit by approximating relationships with polynomials... and investigate "regularization parameters" as a way to avoid overfitting. Template for the work we're doing is contained in the project link for this week. We will use an existing codebase in Jupyter notebooks and change the parameters, add functions, etc.

Final Project!

(Final) Project 12: due Monday, May 20

Congratulations! Send your finals to me by the last day of school before grades are due. I'm quite proud of the work y'all have done this semester.
